PIEZA 1 — Hook usePlanGate
Crea: artifacts/buildos/src/hooks/usePlanGate.ts

Este hook:
- Recibe un error de TanStack Query (de un mutation.error o query.error)
- Detecta si es un error de plan_required (status 403 + body.error === 'plan_required')
- Devuelve: { isPlanError: boolean, feature: string, currentPlan: string, requiredPlan: string }
