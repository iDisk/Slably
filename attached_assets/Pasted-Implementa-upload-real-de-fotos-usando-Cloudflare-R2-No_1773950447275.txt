Implementa upload real de fotos usando Cloudflare R2. 
No agregues funcionalidades fuera de este alcance.

CONTEXTO DEL PROYECTO:
- Backend: Node.js + TypeScript en artifacts/api-server
- Frontend builder: React en artifacts/buildos
- Schema Zod en: lib/api-zod/src/generated/api.ts (archivo manual, NO regenerar)
- Las fotos hoy se guardan con fileUrl como string de URL manual
- Necesito reemplazar eso con upload real desde el navegador/celular

LAS VARIABLES DE ENTORNO YA ESTÁN CONFIGURADAS:
R2_ACCOUNT_ID, R2_ACCESS_KEY_ID, R2_SECRET_ACCESS_KEY, R2_BUCKET_NAME, R2_PUBLIC_URL

---

PASO 1 — Instalar dependencia en el backend:
En artifacts/api-server, instala: @aws-sdk/client-s3 @aws-sdk/s3-request-presigner

---

PASO 2 — Crear el módulo R2 en el backend:
Crea el archivo: artifacts/api-server/src/lib/r2.ts

Debe contener:
- Un cliente S3 configurado con endpoint https://${R2_ACCOUNT_ID}.r2.cloudflarestorage.com
- Credenciales desde process.env
- Región: "auto"
- Una función getPresignedUploadUrl(key: string, contentType: string) que:
  - Genera un PutObjectCommand con el bucket y key
  - Retorna una presigned URL con expiración de 300 segundos (5 minutos)
- Una función getPublicUrl(key: string) que:
  - Retorna ${R2_PUBLIC_URL}/${key}

---

PASO 3 — Nueva ruta en el backend:
En artifacts/api-server/src/routes/uploads.ts crea:

POST /uploads/presigned-url
- Requiere autenticación (JWT válido)
- Body: { fileName: string, contentType: string, projectId: number }
- Validaciones:
  - contentType debe ser uno de: image/jpeg, image/png, image/webp, image/heic
  - fileName no puede estar vacío
  - projectId debe existir y el usuario debe tener acceso (usar checkProjectAccess)
- Genera un key único: photos/${projectId}/${Date.now()}-${randomUUID()}.${extension}
- Retorna: { presignedUrl: string, publicUrl: string, key: string }

Registra esta ruta en app.ts como: app.use('/uploads', uploadsRouter)

---

PASO 4 — Actualizar lib/api-zod/src/generated/api.ts:
Agrega estos dos schemas nuevos al final, antes del último export:

PresignedUrlBody:
- fileName: zod.string().min(1)
- contentType: zod.enum(["image/jpeg", "image/png", "image/webp", "image/heic"])
- projectId: zod.number()

PresignedUrlResponse:
- presignedUrl: zod.string().url()
- publicUrl: zod.string().url()
- key: zod.string()
